PROJECT TITLE - STUDENT RECORD MANAGER(CSV BASED)

DESCRIPTION - DEVELOP A STUDENT INFORMATION MANAGER(GUI) USING PYTHON THAT WOULD DEAL WITH RECORDS IN A CSV FILE. IT WILL BE ABLE TO PERFORM VARIOUS MATHEMATICAL OPERATIONS ON DATA PRESENT, REMOVE DUPLICACY IN RECORDS, AND CHECK THE PRESENCE OF RECORD IN THE CSV FILE. FURTHER GRAPHS WILL ALSO BE IMPLEMENTED ON THE SAME. POSSIBLY A MORE USER FRIENDLY AND INTERESTING INTERFACE WILL BE DEVELOPED.
